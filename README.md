# Clojure-Sleuth

A Clojure clone of the ASCII murder mystery game Sleuth:
http://en.wikipedia.org/wiki/Sleuth_(video_game)

Clojure-sleuth uses the clojure-lanterna library:
http://sjl.bitbucket.org/clojure-lanterna/
